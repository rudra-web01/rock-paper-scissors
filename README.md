✊✋✌️ Rock Paper Scissors Game

An interactive and fun Rock Paper Scissors game built using HTML, CSS, and JavaScript. Play against the computer and track your score in real-time with a clean and responsive UI.

🚀 Features

🎮 Play Rock, Paper, Scissors against the computer

⚡ Instant result display (Win / Lose / Draw)

📊 Real-time score tracking

🎨 Clean and responsive UI design

🔁 Smooth game interactions using JavaScript

🛠️ Tech Stack

HTML5 – Structure

CSS3 – Styling & layout

JavaScript (Vanilla JS) – Game logic & interactivity
